iv. Projects
Project · 01
AquaTrace
AquaTrace is a vessel-tracking system designed to monitor and track maritime activity around the Davao region and its ports. It provides real-time or systematic tracking of ships, helping improve visibility of vessel movements for better monitoring, coordination, and maritime awareness. The system is built to support safer and more efficient port operations by organizing vessel data in a clear and accessible way.
Tracking Systems Monitoring Database
